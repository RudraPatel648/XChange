# XChange

XChange is a simple web-based platform designed to connect people who want to exchange skills and learn from each other. The project focuses on creating a collaborative environment where users can both teach and learn in an interactive way.

---

## 🚀 Project Type
This is a **BWD (Basic Web Development) project** built as part of learning and practicing frontend fundamentals.

---

## 🛠️ Tech Stack
- HTML (Vanilla)
- CSS (Vanilla)

No frameworks or JavaScript libraries are used — the project is built using pure frontend technologies.

---

## ✨ Features
- Clean and modern UI design
- Responsive layout
- Navigation bar and footer
- Hero section with call-to-action
- Team member cards
- Reviews/testimonials section
- Sign In and Sign Up pages

---

## 🎯 Purpose
The goal of this project is to:
- Strengthen understanding of HTML & CSS
- Practice layout design (Flexbox, Grid)
- Build a real-world style landing page
- Improve UI/UX design skills
